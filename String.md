# String 

## Doule quotes 

```javascript 
const doubleQuoteStr = "JavaScript"; 
```

## Single quotes 

```javascript 
let singleQuoteStr = 'JavaScript';
```

## Substitutions 

```javascript 
let count = 0 
console.log('Current count is %s', count); 
```

## Template literals 

```javascript 
const value1 = 10;
const value2 = 23; 
console.log('`value 1 is {value1}` and value 2 is `{value2}`, sum is `{value1 + value2}`');
```
