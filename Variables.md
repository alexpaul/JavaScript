# Variables 

## `const`

> `const` represents a constant variable. This variable cannot be mutated. 

```javascript
const age = 10; 
age = 11; // typeError { Assignment to constant variable. }
```

## `let`

> `let` represents a mutable variable. It can be changed over time. 

```javascript
let year = 2020; 
year = 2021; 
```
